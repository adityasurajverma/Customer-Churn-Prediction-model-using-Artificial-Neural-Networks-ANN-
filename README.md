# Customer-Churn-Prediction-model-using-Artificial-Neural-Networks-ANN-
Optimized ANN model to achieve 77% recall for churn prediction, ensuring minimal loss of high-risk customers.
Excited to share my latest Machine Learning project!

I built a **Customer Churn Prediction model using Artificial Neural Networks (ANN)** to help businesses identify customers likely to leave.

🔍 Key Highlights:
• Achieved **78% test accuracy**
• Achieved **~75% cross-validation accuracy** using K-Fold CV
• Improved **churn recall to 77%** using threshold tuning
• Applied **SMOTE** to handle class imbalance
• Used **GridSearchCV** for hyperparameter optimization

💡 Key Insight:
Instead of focusing only on accuracy, I optimized the model to **maximize recall for churn customers**, ensuring that high-risk customers are not missed.

Final Model Performance:
• Precision (Churn): 0.48
• Recall (Churn): 0.77
• AUC Score: 0.78

🛠 Tools Used:
Python, Keras (TensorFlow), Scikit-learn, NumPy, Pandas

This project helped me understand the importance of balancing **precision vs recall** and building models aligned with **business goals**.
